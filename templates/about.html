About SlamSim

SlamSim is a tool designed to help you manage your simulated wrestling league. It allows you to create wrestlers, tag teams, divisions, belts, and events, and track their progress.

SlamSim! is written in Python using Flask. This application is currently under development. Features like "Fan Mode" and an AI match writing assistant are planned for future releases.

Version: 1.0 Beta 2

Developed by: Gary Hartzell
